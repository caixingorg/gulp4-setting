# gulp4-setting
###本配置适用于gulp4.x

#1.初始化
sudo npm install 或 sudo cnpm install

#2.gulp4的安装

 - 先卸载全局 gulp
 
	$ npm uninstall gulp -g	

 - 安装全局的 gulp 4.0 
 
	$ npm install "gulpjs/gulp#4.0" -g	

 - 到项目目录里删掉本地的 gulp
 
	$ npm uninstall gulp --save-dev	

 - 安装本地的 gulp 4.0
 
	$ npm install "gulpjs/gulp#4.0" --save-dev

#3.启动gulp4.0
	$ sudo gulp
